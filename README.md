# Chanel Multimodal Search

## Description

Projet de recherche de produits Chanel avec texte + image.

## Fonctionnalités

* Recherche texte
* Recherche image
* Recherche combinée

## Lancer le projet

```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```

## Dataset

Chanel Product Prices Germany

## Tech

Python, Streamlit, CLIP, BERT
